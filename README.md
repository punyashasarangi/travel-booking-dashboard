## Project Information

- **Project Name:** Travel Booking Dashboard
- **Project Scope:** To design and develop a modern, responsive, and interactive Travel Booking Dashboard that allows users to search destinations, explore travel packages, and book trips seamlessly across desktop, tablet, and mobile devices.

---

## Features

- Responsive Navigation Bar
- Hero Section with Destination Search Functionality
- Travel Statistics Dashboard
- Popular Destination Cards
- Search Results Page for Destinations
- Destination Package Details
- Trip Booking System
- Booking Confirmation Message
- Newsletter Subscription
- Mobile Hamburger Menu
- Responsive Design for Desktop, Tablet, and Mobile Devices
- Interactive User Interface with Hover Effects and Animations

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Project Pages

### Home Page (index.html)

- Search travel destinations.
- View travel statistics.
- Explore popular destinations.
- Browse featured travel packages.

### Search Results Page (search-results.html)

- Displays destination details based on the user's search.
- Shows destination image, package price, and description.
- Allows users to proceed with booking.

### Booking Page (booking.html)

- Displays the selected destination.
- Allows users to enter booking details.
- Provides booking confirmation upon successful submission.

## How to Run the Project

1. Clone the repository.
2. Open the project folder.
3. Open the project using Live Server in VS Code or any web browser.
4. Open the `index.html` file.
5. Search for a destination.
6. Explore the travel package details.
7. Proceed with trip booking.
8. Confirm the booking successfully.
